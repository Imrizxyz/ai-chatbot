## 🤖 AI Chatbot (V1 → V4)

A Python-based chatbot project showing progression from basic rule-based logic to an AI-powered assistant using OpenAI API.

---

## 🚀 Features
- V1: Basic rule-based chatbot  
- V2: Improved input matching  
- V3: Random smart replies  
- V4: AI-powered chatbot using OpenAI API  

---

## 🛠️ Tech Stack
- Python  
- OpenAI API
